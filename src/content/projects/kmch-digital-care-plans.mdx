## Overview
A comprehensive digital care plan system developed for KMCH. The platform allows healthcare professionals to securely create, edit, and manage patient care plans linked to NHS numbers.

## Architecture
- **Frontend**: React (Create React App) with Redux for state management, tested with Jest.
- **Backend**: Django & Django REST Framework (DRF) serving RESTful APIs as a backend system processing NHS healthcare data securely.
